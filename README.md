# Discord OBS menu

## Installation
```
pip install git+https://github.com/micoli/discord-obs.git
```
or for upgrade
```
pip install --upgrade --force-reinstall git+https://github.com/micoli/discord-obs.git
```

depending of your installation `pip` can be replaced by `pip3`
